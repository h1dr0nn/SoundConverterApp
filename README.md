# Sound Converter App

Ứng dụng Windows để convert audio (MP3, WAV, OGG, FLAC, ...).

## Cài đặt
```bash
pip install -r requirements.txt
```

⚠ Yêu cầu: Cài [ffmpeg](https://ffmpeg.org/download.html) và thêm vào PATH.

## Chạy test nhanh
```bash
python main.py
```

## Build ra .exe
```bash
pyinstaller setup.spec
```
